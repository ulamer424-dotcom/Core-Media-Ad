# Core Media Ad Portfolio

ملفات الموقع جاهزة للرفع مباشرة إلى GitHub ثم الربط مع Cloudflare Pages.

## الملفات
- `index.html`
- `style.css`
- `script.js`
- `projects.json`
- `assets/`

## إضافة مشروع جديد
1. ارفع الفيديو أو الصورة إلى Cloudinary.
2. انسخ رابط الملف.
3. افتح `projects.json`.
4. أضف عنصرًا جديدًا داخل القائمة.
5. اعمل Commit على GitHub.

Cloudflare Pages سيعيد نشر الموقع تلقائيًا.

## إعداد Cloudflare Pages
- Framework preset: None
- Build command: اتركه فارغًا
- Build output directory: `/`
